# User

* **URL**

  _https://api.zxsean.com/user/register_

* **Method:**

  `POST`
  
*  **URL Params**

   **Required:**
 
   `name=[string]` ,
   `email=[string]` ,
   `password=[string]` ,
   `phone=[integer]` 

   **Optional:**
 
   `id=[integer]`

* **Data Params**

    ```json
    {
        "name": "Sean Zhang",
        "email": "sean@randomemail.com",
        "password": "psw123",
        "phone": 2023334444
    }
    ```

* **Success Response:**
  
  * **Code:** 200 <br />
    **Content:** `{ id : 12 }`
 
* **Error Response:**

  * **Code:** 401 UNAUTHORIZED <br />
    **Content:** `{ error : "registration error" }`

------------------------------------------------------------

* **URL**

  _https://api.zxsean.com/user/:id_

* **Method:**

  `GET`
  
*  **URL Params**

   **Required:**
 
   `id=[integer]`

* **Data Params**

    ```json
    {
        "id": 1,
    }
    ```

* **Success Response:**
  
  * **Code:** 200 <br />
    **Content:** `{ id : 11 }`
 
* **Error Response:**

  * **Code:** 401 UNAUTHORIZED <br />
    **Content:** `{ error : "can't find user" }`
